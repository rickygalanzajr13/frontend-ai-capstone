# Frontend AI Engineering Capstone

This repository contains my capstone project for the Front-end AI Engineering track.

## Goal

Build a modern, responsive web application while applying AI-assisted development workflows.

## AI-Assisted Development

This capstone emphasizes working with AI tools throughout the build process—not just for code generation, but for planning, iteration, and review. Practices include:

- Using AI assistants for scaffolding, refactoring, and documentation
- Reviewing and testing all AI-generated code before committing
- Documenting key decisions and the prompts that shaped them
